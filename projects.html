    UCSB Polls

    Created a webapp which allows anyone with a UCSB email to vote on virtually anything. Any user can search, create, and vote on polls (and view the results after voting). The user interacts with a ReactJS front end Webapp. The ReactJS front end interacts with Auth0 and an Express server (hosted on localhost or heroku) to authenticate and fetch users. The server interacts with a Firebase database to get and update information about polls and users. I developed this project along with a team of other students using Agile workflows (standups, sprints, etc.)

    May 2021 - CMPSC 148

    SigNN - Real-time ASL Alphabet Translator

    SigNN stands for Sign Neural Network, and it is built upon MediaPipe, a Machine-Learning Pipeline made by Google. SigNN is the first app which can translate ASL (American Sign Language) in real-time. It shows that real-time sign language translation is now possible on mobile devices. SigNN was developed by a group of students at the University of California, Santa Barbara.

    2020 - SigNN Team / UCSB Data Science

    Password Vault

    Password Vault is a desktop program with a graphical interface capable of encrypting and decrypting user-provided passwords. Python scripts handle encrpytion key generation, encrypting, saving, loading, and decrypting passwords with a saved key. All encryption is 256-bit AES.

    September 2020

    Construction Charlie

    Construction Charlie is a video game for Windows featuring action-packed platforming gameplay. The game features several detailed environments, different enemy types, and a bundled level editor, which lets users generate their own levels to load into the game. The game was developed in Java, using the Swing graphics library. As the only programmer on the development team, I diligently added new features to Construction Charlie, culminating in its release on the popular desktop gaming platform Steam.

    September 2019

    Cloudflu

    CloudFlu is an application developed at the BASEHacks hackathon which analyses tweets and CDC data (gathered using the SickWeather API) for signs of influenza or other major outbreaks.
